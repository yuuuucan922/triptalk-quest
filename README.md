# TripTalk Cafe

旅行英会話をカフェエリアから育てていく学習ゲームです。

## Features

- 英単語モード
- 英文モード
- 英会話ロールプレイモード
- XP によるカフェ内装アップグレード
- XP によるバリスタの服装・肩書き変化
- Phrase bank に表現を保存

## GitHub Pages

GitHub Pages ではリポジトリ直下を公開します。

```text
Settings -> Pages -> Deploy from a branch
Branch: main
Folder: /root
```

公開後の URL は次の形式です。

```text
https://<github-user-name>.github.io/<repository-name>/
```

## Local Preview

```powershell
node serve-local.js 5173
```

Then open:

```text
http://localhost:5173/
```
